# <a name="skype-for-business-online-service-description"></a>商務用 Skype Online 服務說明

> [!NOTE]
> Microsoft Teams 現在是 Office 365 中的郵件、會議及通話的主要用戶端。 從2018年10月1日起，，將會架 Microsoft Teams 至具有500座位的新客戶，也無法存取商務用 Skype 線上。 到2019年9月1日為止，這適用于所有新的客戶。 已在線上使用商務用 Skype 的承租人可以繼續進行， (包括布建新使用者) ，直到他們完成對 Microsoft Teams 的轉換為止。

> [!NOTE]
> 商務用 Skype線上方案2于2019年7月1日從銷售退休。 這表示您將無法再取得立即訊息、目前狀態、會議和對等語音和影片商務用 Skype 線上方案2。 所有包含 Teams 的計畫都會繼續支援這些功能，包括 Office 365 套件。 當您的計畫在您目前的字詞結束時到期時，為了能夠繼續存取這些功能，您必須切換至不同的計畫。 建議您評估目前已指派商務用 Skype Online Plan 2 之使用者的 Microsoft 365 商務基本版、商務進階版或任何其他 Office 套件。 

 **您是否要知道您需要購買哪些授權才可取得特定的商務用 Skype 功能？請參閱 [商務用 Skype 附加元件授權](/SkypeForBusiness/skype-for-business-and-microsoft-teams-add-on-licensing/skype-for-business-and-microsoft-teams-add-on-licensing)。**
  
商務用 Skype線上是一種通訊服務，可隨時隨地線上會議和交談的人員。 它讓使用者可以存取目前狀態的資訊、啟用即時訊息、音訊和視訊通話、豐富的線上會議，以及多樣的 Web 會議功能。 
  
商務用 Skype 是裝載於多租用戶伺服器上，同時支援多個客戶。 這些伺服器是存放在 Microsoft 資料中心，可供使用者從公司網路內部或透過網際網路，存取範圍廣泛的裝置。
