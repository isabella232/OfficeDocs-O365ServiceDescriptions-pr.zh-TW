# <a name="excel-for-the-web"></a>Excel 網頁版

Excel 網頁版 (過去 Excel 的 web App) 會將您的 Microsoft Excel 體驗延伸至網頁瀏覽器，您可以在其中儲存活頁簿的網站上直接使用活頁簿。 所有客戶皆可使用 Office 網頁版來查看及輕微編輯 Office 檔案。
  
> [!NOTE]
> 在 Microsoft Office SharePoint Online 內無法從 Excel 網頁版中查看超過 50 mb (mb) 的活頁簿。 若要檢視較大的檔案，則您的電腦需要安裝 Excel 桌面應用程式。 

> [!NOTE]
> 為了協助您比較服務，本文所列的高級功能包括只有 Office 套件中提供的 Excel 桌面應用程式功能 (例如 Office 專業增強版、Office 標準版或 Microsoft 365 Apps 企業版) 。 如需所有 Office 網頁版功能（如 Word 網頁版和 PowerPoint 網頁版）的清單，請參閱[不同 Office 網頁版計畫中的功能可用性](office-online-service-description.md#feature-availability-across-office-for-the-web-plans)。

若要查看 Excel 網頁版所支援的檔案類型，請參閱[Office 網頁版支援的檔案類型](office-online-service-description.md#supported-file-types-for-office-for-the-web)。 若要深入瞭解 Excel 網頁版和 Excel 桌面應用程式之間的差異，請參閱在[瀏覽器中和 Excel 中使用活頁簿的差異](https://support.office.com/article/f0dc28ed-b85d-4e1d-be6d-5878005db3b6)。
  
## <a name="add-background-color-to-sheet-tabs"></a>新增背景色彩至工作表索引標籤 

將背景填滿色彩新增至活頁簿底部的工作表索引標籤，使其醒目。如需詳細資訊，請參閱 [將背景色彩新增至工作表](https://support.office.com/article/440B28F2-3146-4DCA-95DF-3B9D43ACBE59)索引標籤。

## <a name="add-hyperlinks"></a>新增超連結

在儲存格中新增超連結可連結至網頁，或在網頁上開啟檔案。您可以在儲存格中直接輸入網址，或使用 [插入超連結] 命令。

## <a name="advanced-data-types-stocks-and-geography"></a>高級資料類型：股票和地理位置

您可以在 Excel 中取得股價和地理資料。 在儲存格中輸入文字，然後將其轉換成股票資料型別或 Geography 資料型別很簡單。 深入瞭解 [股票和地理資料類型](https://support.office.com/article/e61a33056-9935-484f-8ac8-f1a89e210877)。
  
## <a name="advanced-time-filtering-timeline-slicer"></a>進階時間篩選 (時間表交叉分析篩選器)

「時間表」控制可讓您篩選模型化資料。視覺化的呈現方式可讓您檢視和變更日期的持續範圍，以及篩選樞紐型物件，例如：樞紐分析表和樞紐分析圖。  
  
## <a name="alignment"></a>對齊

使用 [首頁] 索引標籤上的 [對齊] 按鈕來變更文字的對齊方式。  
  
## <a name="apply-conditional-formatting"></a>套用條件格式化設定

使用條件化格式可協助您以視覺方式流覽及分析資料、偵測重大問題，以及識別模式和趨勢。 深入瞭解如何[在 Excel 網頁版中使用設定格式化的條件](https://support.microsoft.com/office/use-conditional-formatting-to-highlight-information-fed60dfa-1d3f-4e13-9ecb-f1951ff89d7f?)。
  
## <a name="apply-data-validation-to-cells"></a>將資料驗證套用至儲存格

Excel 網頁版中提供此功能。 其他的高級功能只能在 Excel 桌面應用程式中使用。
  
## <a name="apply-smartrecommended-formatting"></a>套用智慧/建議的格式

在 Excel 網頁版中無法使用。 在 Excel 桌面應用程式中，建議的圖表可讓您挑選最適合用來呈現資料的各種圖表。 [若要深入瞭解如何使用 Excel 中的圖表建議](https://support.office.com/article/cd131b77-79c7-4537-a438-8db20cea84c0)。
  
## <a name="autocomplete"></a>自動

Excel 網頁版可以完成您在儲存格中輸入的內容，而且如果有一個以上的可能值，請在您可以選擇的清單中顯示所有值。
  
## <a name="autosum"></a>AutoSum

選取您要加總的儲存格，然後按兩下 [自動加總]，即可計算某欄或某列的總數。結果將顯示在下一個空白儲存格中。
  
## <a name="calculations"></a>計算

您可以使用 Excel 來執行各種自動、手動與反覆運算。Excel 會使用函數來執行這些計算。如需詳細資訊，請參閱 [Excel 函數 (依類別)](https://go.microsoft.com/fwlink/?linkid=847459)。

## <a name="cell-references-in-formulas"></a>公式中的儲存格參照

您可以在公式中使用儲存格參照。 當您 [參照公式中的儲存格或範圍](https://support.office.com/article/c7b8b95d-c594-4488-947e-c835903cebaa)時，參照的儲存格或範圍會以色彩反白顯示。 如需詳細資訊，請參閱 [使用公式中的儲存格參照](https://support.office.com/article/fe137a0d-1c39-4d6e-a9e0-e5ca61fcba03)。
  
## <a name="chart-animations-adapt-to-new-data"></a>圖表動畫可隨新資料改變
僅 Microsoft Excel 桌面應用程式支援進階圖表功能。進一步了解[如何製作 SmartArt 圖形動畫](https://go.microsoft.com/fwlink/p/?LinkId=271668)。
  
## <a name="charts-and-tables-including-pivotchart-reports-and-pivottable-reports"></a>圖表與表格 (包含樞紐分析圖和樞紐分析表)

Excel 圖表與表格可讓您輕鬆格式化資料和管理資訊。 若要深入瞭解，請參閱[建立及格式化表格](https://support.office.com/article/e81aa349-b006-4f8a-9806-5af9df0ac664)，並[PivotTable 和 PivotChart 報告的概覽](https://go.microsoft.com/fwlink/?linkid=55417)。

## <a name="check-spelling"></a>檢查拼寫

若要檢查工作表上任何文字的拼寫，請選取 [**檢查**  >  **校對**  >  **拼寫**]。 深入瞭解 [檢查拼寫](https://support.office.com/article/185efb8d-ee3f-4511-b7ef-35599c924159)。

## <a name="comment-creation"></a>建立註解

您可以在 Excel 網頁版中插入、編輯和刪除批註。 您也可以解析批註執行緒，它會將它關閉以進行編輯及新增至新的批註，但會將它保留在原地。 這可讓您視需要將螺紋重新開啟為批註。 [在您的批註中使用 @mentions](https://support.office.com/article/644bf689-31a0-4977-a4fb-afe01820c1fd?ad=US#ID0EAADAAA=Online) ，將電子郵件傳送給您提及的人員。 進一步了解[在 Excel 中新增註解](https://go.microsoft.com/fwlink/p/?LinkId=271670)。

## <a name="convert-a-table-to-a-range"></a>將資料表轉換成範圍

在您建立 Excel 表格之後，您可能只會想要沒有表格功能的表格樣式。 若要停止使用表格中的資料，而不會遺失任何已套用的表格樣式格式，您可以將此表格轉換成工作表中的一般資料範圍。 深入瞭解如何[將 Excel 表格轉換成資料範圍](https://support.office.com/article/0b326ff1-1764-4ebe-84ea-786265d41c77#ID0EAADAAA=Web)。
  
## <a name="copy-and-paste"></a>複製和貼上

您可以剪下、複製及貼上文字、超連結、數位、公式、圖形、圖表及影像。 如果資料行中的資料必須旋轉以重新排列列，請使用 [**貼** 上] 功能表上的 [**換位**] 功能。 深入瞭解[Excel 網頁版中的複製和貼](https://support.office.com/article/682704da-8360-464c-9a26-ff44abf4c4fe#bmk-excelonline)上。 以您的瀏覽器使用 Edge 或 Chrome，您可以複製其他應用程式的超連結，並將其貼到 Excel 網頁版。 貼上的超連結會正常運作。
  
## <a name="create-external-data-connections"></a>建立外部資料連結

您可以使用 Excel 網頁版來查看資料連線，但無法使用 Excel 網頁版建立外部資料連線。 您將需要 Excel 桌面應用程式才能使用外部資料。
  
## <a name="create-tables"></a>建立表格

建立表格以組織和分析相關資料。 表格便於進行排序、篩選和格式化資料。 使用 [表格格式設定選項](https://support.office.com/article/format-an-excel-table-6789619f-c889-495c-99c2-2f971c0e2370)（包括樣式庫）新增一些波蘭文。
  
## <a name="creation-of-advanced-analysis-views-power-view-power-pivot-slicers"></a>建立進階分析檢視 (Power View、Power Pivot、交叉分析篩選器)

僅 Excel 桌面應用程式提供進階分析檢視 (例如：Power Pivot 和 Power View) 功能。 Excel 網頁版不支援這些功能。 進一步了解 [PowerPivot](https://go.microsoft.com/fwlink/p/?LinkId=271671)。
  
## <a name="data-validation"></a>資料驗證

您可以使用資料驗證來限制資料或使用者輸入儲存格的值類型。 若要深入瞭解，請參閱 [將資料驗證套用至儲存格](https://go.microsoft.com/fwlink/?linkid=844735)。
  
## <a name="drag-and-drop-cells"></a>拖放儲存格

您可以將儲存格從某處拖曳至另一個位置，以在 Excel 網頁版中移動儲存格的值。

## <a name="draw-cell-borders"></a>繪製儲存格框線

您可以在工作表上的儲存格周圍繪製框線，以協助您以視覺方式組織資料。 深入瞭解[Excel 中的儲存格框線](https://support.office.com/article/dc8a310b-92e3-46a7-9f17-2ab745810f4a#ID0EAADAAA=Web)。
  
## <a name="dropbox"></a>Dropbox

Dropbox 是一個檔案主控服務，提供雲端儲存體、檔案同步作業、個人雲端及用戶端軟體。
  
## <a name="duplicate-copy-worksheets"></a>重複的 (複製工作表) 

您可以重複 (或複製活頁簿或 Excel 網頁版中的其他活頁簿中的) 工作表。 深入瞭解[在 Excel 網頁版中複製工作表](https://support.office.com/article/47207967-bbb2-4e95-9b5c-3c174aa69328#ID0EAACAAA=Web)。

## <a name="embed-workbook-on-web-or-blog-page-onedrive"></a>在網頁或博客頁面上嵌入活頁簿 (OneDrive) 

如果您將 Excel 活頁簿儲存在 OneDrive.com 中，則可在部落格或網站上直接嵌入活頁簿。您的讀者可以直接在該處排序、篩選和計算資料，且若您在 OneDrive 中更新活頁簿內容，他們下次重新整理頁面時也會看到最新的變更。進一步了解[在部落格中嵌入 Excel 活頁簿](https://go.microsoft.com/fwlink/p/?LinkId=271673)。
  
## <a name="external-references-links"></a>外部參考 (連結)

外部參考 (也稱為連結) 是其他 Excel 活頁簿中工作表上的儲存格或範圍的參考，或是另一個活頁簿中所定義名稱的參考。 您可以使用 Excel 網頁版來查看外部參照，但無法使用 Excel 網頁版建立。 您將需要 Excel 的桌面應用程式，才能建立或更新外部參照。 
  
## <a name="fill-handle"></a>填滿控點

拖曳所選儲存格右下角的填滿控點，即可將資料填入相鄰的儲存格。
  
## <a name="find"></a>尋找

在使用中的工作表中尋找儲存格內容。 [簡易使用] 對話方塊可讓您從工作表中目前的選取範圍向上或向外搜尋的選項。 [**尋找全部**] 可讓您反白顯示搜尋字詞的每個實例。
  
## <a name="font-and-cell-formatting"></a>字體和儲存格格式化

根據您的需求自訂您的資料樣式。 從各式字體樣式或顏色中進行選擇，或變更文字的大小和顏色。 此外，您也可以繪製儲存格框線，並選取框線色彩。
  
## <a name="formula-bar"></a>資料編輯列

請參閱資料編輯列中儲存格結果後方的公式。 您可以在瀏覽器中新增、變更和刪除部分公式，就如同在 Excel 桌面應用程式中操作一樣。 Excel 網頁版會使用可在公式與工作表上的資料之間提供視覺對應的色彩，強調公式的引數。
  
## <a name="formula-tools-advanced"></a>公式工具，高級

Microsoft Excel 桌面應用程式提供最高級的公式工具，例如3d 參考樣式。 Excel 網頁版支援數量不斷增加的高級 Excel 公式，例如[動態陣列公式](https://support.microsoft.com/office/dynamic-array-formulas-and-spilled-array-behavior-205c6b06-03ba-4151-89a1-87a7eb36e531)。 進一步了解[公式](https://support.microsoft.com/office/ecfdc708-9162-49e8-b993-c311f47ca173)。
  
## <a name="freeze-panes"></a>凍結窗格

「凍結窗格」功能可在 [Excel 網頁版] 功能區中取得。 深入瞭解[在 Excel 中凍結窗格的運作方式](https://go.microsoft.com/fwlink/p/?LinkId=282331)，並[查看 Excel 網頁版的秘訣](https://go.microsoft.com/fwlink/?linkid=869873)。
  
## <a name="full-fidelity-reading-view"></a>高逼真讀取檢視

[讀取檢視] 中顯示的活頁簿內容和 [編輯檢視] 的瀏覽器中顯示的內容完全相同。
  
## <a name="functions"></a>函式

您可以在 Excel 網頁版中使用大部分超過 400 Excel 的工作表函數，包括在[動態陣列公式](https://support.microsoft.com/office/dynamic-array-formulas-and-spilled-array-behavior-205c6b06-03ba-4151-89a1-87a7eb36e531)中使用的函數，例如[FILTER 函數](https://support.office.com/article/filter-function-f4f7cb66-82eb-4767-8f7c-4877ad80c759)。
  
## <a name="goto"></a>GoTo

使用「移至」功能，快速導覽您的試算表。 請在儲存格參照中輸入位置，並跳至試算表中的那個位置。 深入瞭解[Excel 網頁版中的鍵盤快速鍵](https://go.microsoft.com/fwlink/p/?LinkID=272943)。
 
## <a name="group-data"></a>群組資料

您可以在 Excel 網頁版試算表中群組或大綱列與欄。 鍵盤快速鍵可讓您輕鬆地快速展開或折迭您建立的群組。 深入瞭解[Excel 網頁版中的資料分組](https://support.office.com/article/08ce98c4-0063-4d42-8ac7-8278c49e9aff#ID0EAADAAA=Web)。
 
## <a name="hideunhide-rows-columns-and-sheets"></a>隱藏/取消隱藏列、欄和工作表

您可以在 Excel 網頁版中隱藏和取消隱藏活頁簿中的列、欄和工作表。

## <a name="insert-pictures"></a>插入圖片

在試算表中插入圖片，以新增一些視覺意義。 深入瞭解[在 Excel 網頁版中插入圖片](https://support.office.com/article/73660701-018b-4961-bc28-6976cf9ebfb3)。

## <a name="insert-shapes"></a>插入圖形

在您的 Excel 網頁版試算表中新增及格式化圖形，例如方塊、圓形、線條、連接器或箭號。 深入瞭解 [新增圖案](https://support.office.com/article/0e492bb4-3f91-43b5-803f-dd0998e0eb89#OfficeVersion=Web)。

## <a name="keyboard-shortcuts"></a>快速鍵

您可以控制鍵盤快速鍵，使其在桌面和 web 版本的 Excel 中的運作方式相同。您可以變更快速鍵設定 &mdash; ，甚至可以覆寫瀏覽器快捷方式。 請選取 [**説明**  >  **快速鍵**]。
  
## <a name="merge-cells"></a>合併儲存格

Use Merge &amp; Center on the Home tab to combine and center the contents of the selected cells in one larger cell. 您可以選取對齊按鈕來變更對齊方式。

## <a name="named-ranges"></a>命名範圍

您無法在 Excel 網頁版中建立命名範圍，但是您可以使用在 Excel 網頁版的試算表的 Excel 桌面中所建立的命名範圍。 當您選取一個命名範圍時，該名稱會顯示在公式欄的上方。

## <a name="number-formatting"></a>數字格式化

變更數字的格式。從格式清單中挑選格式 (如：貨幣或簡短日期)、變更小數位數或插入千位分隔符號。
  
## <a name="office-add-ins"></a>Office 增益集

 Excel 網頁版中提供某些部分，但並非所有 Office 增益集。 您可以從 Office 市集取得 Excel 增益集。 進一步了解[如何取得 Excel 增益集](https://go.microsoft.com/fwlink/p/?LinkId=271667)。

## <a name="offline-viewing-and-authoring"></a>離線檢視和撰寫

從網際網路瀏覽器啟動 Excel 網頁版，並依賴網際網路連線。 若要離線存取試算表，您必須在電腦上安裝 Microsoft Excel 桌面應用程式，以用來檢視和編輯 Excel 試算表。

## <a name="pivottables"></a>樞紐分析表

您可以在 Excel 網頁版試算表中插入 PivotTables，並計算、匯總及分析資料。 深入瞭解如何[在 Excel 網頁版中插入 PivotTable](https://support.office.com/article/A9A84538-BFE9-40A9-A8E9-F99134456576#OfficeVersion=Web)。

## <a name="post-to-social-network-and-present-online"></a>發佈到社交網路並於線上呈現

您可以將所選的試算表內容嵌入社交網路頁面或商務用 Skype 對話或會議中，即可和其他使用者共用選取的部分。 Excel 網頁版不支援這些功能。
  
## <a name="power-pivot-viewing"></a>Power Pivot 檢視

Excel 網頁版可讓您查看 power pivot 表格和圖表，但是您必須具有 Excel 的桌面應用程式，才能建立 power pivot 資料模型。
  
## <a name="power-view-viewing"></a>Power View 檢視

您可以在活頁簿中查看 power view 工作表並與其互動，但無法使用 Excel 網頁版建立 power view。 只有使用桌面應用程式才能建立 Power View。 Power View 可讓您在互動式、能隨時呈現的檢視中，建立、共用及瀏覽資料。 在單一工作表上相互連結的互動式地圖、圖表和表格中檢視資訊。 進一步了解 [Power View](https://go.microsoft.com/fwlink/p/?LinkId=271674)。
  
## <a name="print"></a>列印

使用 Excel 網頁版中的 [列印] 命令，將整個工作表或目前的選取範圍傳送至印表機。 深入瞭解[如何在 Excel 網頁版列印](https://go.microsoft.com/fwlink/p/?LinkId=271676)。
  
## <a name="real-time-co-authoring"></a>即時共同撰寫

只要使用網頁瀏覽器 (而非 Excel 桌面應用程式) 開啟檔案，即可讓兩位以上的使用者同時編輯同一試算表。 即時目前狀態可讓您看見共同撰寫作者正在文件何處進行編輯，讓您避免在編輯時與之衝突，並且您可以看見正在進行的變更。 如需即時共同撰寫的詳細資訊，請參閱在[共同撰寫的同時 Excel 活頁簿進行](https://go.microsoft.com/fwlink/?linkid=850793)共同作業。
  
## <a name="recommended-chart-creation-and-editing-with-formatting-controls"></a>使用格式化控制項來建立和編輯建議的圖表

僅 Microsoft Excel 桌面應用程式支援進階圖表功能。 Excel 網頁版不支援這些功能。 進一步了解[可用的圖表類型](https://go.microsoft.com/fwlink/p/?LinkId=271666)。

## <a name="remove-duplicate-values"></a>移除重複值

您可以移除範圍或表格中的重複值。 如需詳細資訊，請參閱 [篩選唯一值或移除重複值](https://support.office.com/article/ccf664b0-81d6-449b-bbe1-8daaec1e83c2)。
  
## <a name="rename-and-add-sheets"></a>重新命名和新增工作表

選取新的工作表圖示以新增工作表。 用滑鼠右鍵按一下任何工作表索引標籤，即可對其重新命名。
  
## <a name="rename-file-while-workbook-is-open"></a>在活頁簿開著時重新命名檔案

在 Excel 網頁版中，您可以在不關閉活頁簿檔的情況下，在標題中選取檔案名，然後輸入新名稱，以重新命名活頁簿檔案。 若要使用 Excel 桌面應用程式重新命名檔案，則需要關閉檔案、瀏覽至其在裝置上的位置，然後重新命名檔案。
  
## <a name="replace"></a>取代

在 Excel 網頁版中無法使用。 如果您的電腦已安裝 Excel 桌面應用程式，則您可以使用 Excel 來尋找和取代文件中的內容。 深入瞭解[使用 Excel 中的 [尋找及取代](https://go.microsoft.com/fwlink/p/?LinkId=282333)]。
  
## <a name="rights-management-irm-and-password-security"></a>版權管理：IRM 和密碼安全性

 在 Excel 桌面應用程式中，您可以使用密碼、權限和其他限制方式保護活頁簿的資料安全。 在 Excel 網頁版中，您可以載入包含工作表保護的活頁簿，並與其互動，以防止使用者選取或輸入受保護的儲存格。 進一步了解[保護 Excel 活頁簿的資料安全](https://go.microsoft.com/fwlink/p/?LinkId=271672)。
  
## <a name="save-or-download-a-copy"></a>儲存或下載副本

需要儲存活頁簿副本嗎？使用 [檔案] 索引標籤上的 [另存新檔] 命令，或者用滑鼠右鍵按一下資料夾中的活頁簿名稱，然後使用 [下載] 命令傳送副本至您的電腦。

## <a name="sensitivity-labels"></a>敏感度標籤

手動套用標籤或使用 Microsoft 的自動建議標籤，以維持對檔的存取和敏感度的控制。
  
## <a name="share"></a>共用

只要按一下滑鼠，就可以把連結傳送給其他使用者，以供其檢視或編輯文件。 深入瞭解[使用 Office 網頁版在 Office 365 中共同作業](https://go.microsoft.com/fwlink/p/?linkid=272770)。
  
## <a name="sheet-protection"></a>工作表保護

當您與其他使用者共用 Excel 檔案時，您可以保護工作表，以協助防止變更；您可以選擇允許您的工作表使用者可執行的動作。

## <a name="sheet-views"></a>工作表視圖

[工作表視圖](https://support.microsoft.com/office/sheet-views-in-excel-0eea3dc5-d7d1-44c5-a953-25ebfbd6c1a6)可讓您建立 Excel 工作表的自訂視圖，而不會被其他人中斷。 例如，您可以設定篩選，只顯示對您很重要的記錄，而不會受到檔中的其他排序和篩選的影響。 您甚至可以在相同的工作表上設定多個工作表視圖。
  
## <a name="slicers"></a>交叉分析篩選器

在工作表中查看交叉分析篩選器，並選取 [交叉分析篩選] 按鈕以篩選資料。 您無法在 Excel 網頁版中建立或編輯切片器，但可刪除現有的交叉分析篩選器。
  
## <a name="sort-and-filter-data"></a>排序和篩選資料

在 Excel 網頁版中，您可以查看和重新排序所有工作表、排序及篩選資料 (從上到下或從左至右，多個層次) ，然後深入瞭解 PivotTables 的詳細資料。 使用圖示集或色彩值，排序使用圖示集或色彩比例的設定條件格式的資料。 深入瞭解 [排序和篩選資料](https://support.office.com/article/62d0b95d-2a90-4610-a6ae-2e545c4a4654?ad=US#ID0EAADAAA=Web)。
  
## <a name="spreadsheet-audit-and-compliance"></a>試算表審核和法規遵循

在 Microsoft Excel 桌面應用程式中，您可以使用工作表的查詢和比較功能來比較活頁簿版本、分析活頁簿的問題或不一致情況，或查看活頁簿和工作表之間的連結。 Excel 網頁版不支援這些功能。 進一步了解[如何使用試算表查詢功能](https://go.microsoft.com/fwlink/p/?LinkId=271677)。
  
## <a name="status-bar-aggregates"></a>狀態列彙總

當您在 Excel 網頁版中選取一組儲存格時，您可以在狀態列中看到 SUM、AVERAGE 和 COUNT。 您也可以選擇想要查看的彙總來自訂狀態列。
  
## <a name="surveys"></a>問卷調查

使用調查將 Excel 網頁版表單傳送給他們可以填寫的使用者。回應將會自動在線上工作表中編譯。 深入瞭解[Excel 中的調查](https://go.microsoft.com/fwlink/p/?LinkId=391470)。
  
## <a name="tell-me"></a>操作說明搜尋

當您需要在 Excel 網頁版中完成某項作業，但是不知道如何執行時，您可以使用「告訴我搜尋」搜尋功能，快速找出您要尋找的專案。 「操作說明搜尋」了解您正在嘗試完成的作業，並且會提供建議，協助您更快速地完成作業。

## <a name="text-formatting"></a>文字格式

您可以套用各種文字格式設定選項，包括刪除線、增加/減少縮排 Excel 網頁版。

## <a name="total-data"></a>資料總數

您可以透過啟用 [切換匯總列] 選項，快速地匯總資料表中的資料。 如需詳細資訊，請參閱[合計 Excel 表格中的資料](https://support.office.com/article/6944378F-A222-4449-93D8-474386B11F20#ID0EAADAAA=Web)。

## <a name="undo-and-redo"></a>復原和取消復原

Excel 網頁版會自動儲存您的工作。 如果您犯了錯誤，請使用 [復原] 或 [按 Ctrl+Z]，然後按 Ctrl + Y。 深入瞭解[Excel 網頁版中的鍵盤快速鍵](https://go.microsoft.com/fwlink/p/?LinkId=272943)。
  
## <a name="vba-and-macro-scripting"></a>VBA 和巨集指令碼

Microsoft Excel 桌面應用程式是功能相當強大的工具，可用於操控、分析和呈現資料。 儘管有這麼多功能，有時候您的組織可能會發現使用 Visual Basic for Applications (VBA) 這種程式語言來執行操作更簡單，透過 VBA 建立巨集可執行一般的重複性工作，或執行使用者介面 (UI) 無法解決的部分工作。 您無法在 Excel 網頁版中使用 vba 建立宏，但是可以開啟和編輯已啟用 VBA 功能的試算表，而不移除 (或損壞) 檔中所包含的 VBA。 進一步了解[開始使用巨集](https://go.microsoft.com/fwlink/p/?LinkId=271678)。
  
## <a name="view-3d-charts"></a>檢視 3D 圖表

在您的工作表中檢視 3D 圖表。 您無法在 Excel 網頁版中建立或編輯3d 圖表。
  
## <a name="what-if-analysis-tools"></a>What if 分析工具

在 Excel 網頁版中無法使用。 如預期所示，您可以視情況來查看 Excel 網頁版的「假設」分析結果。 如果您想使用分析工具 (例如：目標搜尋、資料表、規劃求解和數列)，則您需要 Excel 桌面應用程式。 進一步了解[在瀏覽器中和 Excel 桌面應用程式中使用活頁簿的差異](https://go.microsoft.com/fwlink/p/?LinkId=273900)。

活頁簿統計資料功能會統計及識別活頁簿的元素，以協助您探索該活頁簿的所有內容。 您可以在 [ **回顧** ] 索引標籤上找到它。
